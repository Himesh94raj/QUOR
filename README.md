# 🚀 About QUOR

QUOR is India's decentralized marketplace built to transform creator marketing through secure, performance-driven collaborations.

The platform connects creators and brands with professional video clippers, enabling high-impact short-form marketing campaigns while ensuring transparent payouts based on verified organic performance.

By combining secure campaign management, escrow protection, and analytics, QUOR creates a trusted ecosystem where creators pay only for real results and clippers earn fairly for quality content.

# ⚙️ How It Works

### 1️⃣ Create Campaign

Creators launch a campaign by defining objectives, rewards, and campaign requirements.

↓

### 2️⃣ Join Campaign

Professional clippers browse and participate in available campaigns.

↓

### 3️⃣ Publish Content

Clippers create engaging short-form videos using campaign assets.

↓

### 4️⃣ Performance Verification

Campaign performance is securely verified before rewards become eligible.

↓

### 5️⃣ Reward Distribution

Qualified participants receive transparent performance-based payouts.

# 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/yourusername/QUOR.git
```

### Install Dependencies

```bash
npm install
```

### Configure Environment

Create a `.env` file based on `.env.example`.

```env
PORT=
DATABASE_URL=
AUTH_PROVIDER=
API_KEYS=
```

### Start Development

```bash
npm run dev
```

### Production Build

```bash
npm run build
```
